# IAS Zone Cluster (0x0500)

## 概述

IAS Zone Cluster（Cluster ID `0x0500`）属于 ZCL Rev 7 的 Security and Safety 功能域，章节定位为 8.2。定义入侵报警系统 zone 设备的状态、类型、enroll 和状态变更上报。

Rev 7 精读范围：`raw/specs/07-5123-07-ZigbeeClusterLibrary_Revision_7-1.pdf p.500-p.511`。本页根据这些页的 cluster identifier、overview、server/client、attributes、commands 和行为说明整理；长表字段仍以来源页为准。

### 规范摘要

Please see Chapter 2 for a general cluster overview defining cluster architecture, revision, classification, identification, etc. The IAS Zone cluster defines an interface to the functionality of an IAS security zone device. IAS Zone supports up to two alarm types per zone, low battery reports and supervision of the IAS network.
