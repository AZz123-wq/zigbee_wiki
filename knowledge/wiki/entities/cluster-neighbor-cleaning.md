# Neighbor Cleaning Cluster (0x0023)

## 概述

Neighbor Cleaning Cluster（Cluster ID `0x0023`）属于 ZCL Rev 7 的 Retail 功能域，章节定位为 14.4。管理移动设备邻居清理相关接口。

Rev 7 精读范围：`raw/specs/07-5123-07-ZigbeeClusterLibrary_Revision_7-1.pdf p.897-p.900`。本页根据这些页的 cluster identifier、overview、server/client、attributes、commands 和行为说明整理；长表字段仍以来源页为准。

### 规范摘要

Please see Chapter 2 for a general cluster overview defining cluster architecture, revision, classification, identification, etc. This cluster provides an interface to enable the management of mobile devices in a network. If a stack supports neighbor entry aging, the mobile device will be able to use this cluster to clean the information in the parent/neighbor. An application will be able to configure the aging timeout greater than a KeepAliveTime (attribute supported by a mobile device)...

### 依赖

This cluster should be supported by devices that are acting as routers for Mobile devices in the network; besides, the mobile devices within the network infrastructure (e.g., Hand Held devices or Mobile phones) should use the Mobile Device Configuration Cluster to take advantage of the mobility feature.
