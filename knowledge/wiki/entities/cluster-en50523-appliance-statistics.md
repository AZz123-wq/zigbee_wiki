# EN50523 Appliance Statistics Cluster (0x0B03)

## 概述

EN50523 Appliance Statistics Cluster（Cluster ID `0x0B03`）属于 ZCL Rev 7 的 Appliance 功能域，章节定位为 15.5。提供电器统计日志和统计项读取接口。

Rev 7 精读范围：`raw/specs/07-5123-07-ZigbeeClusterLibrary_Revision_7-1.pdf p.923-p.929`。本页根据这些页的 cluster identifier、overview、server/client、attributes、commands 和行为说明整理；长表字段仍以来源页为准。

### 规范摘要

This cluster provides a mechanism for the transmitting appliance statistics to a collection unit (gateway). The statistics can be in format of data logs. In case of statistic information that will not fit the single payload, the Partition cluster SHOULD be used. Each appliance uses persistent memory to temporarily store collected statistic logs (entries). The maximum number of stored statistic logs is appliance dependent.
