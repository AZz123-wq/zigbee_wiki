# Occupancy Sensing Cluster (0x0406)

## 概述

Occupancy Sensing Cluster（Cluster ID `0x0406`）属于 ZCL Rev 7 的 Measurement and Sensing 功能域，章节定位为 4.8。配置 occupancy sensor，并报告占用状态。

Rev 7 精读范围：`raw/specs/07-5123-07-ZigbeeClusterLibrary_Revision_7-1.pdf p.316-p.322`。本页根据这些页的 cluster identifier、overview、server/client、attributes、commands 和行为说明整理；长表字段仍以来源页为准。

Please see Chapter 2 for a general cluster overview defining cluster architecture, revision, classification, identification, etc. The server cluster provides an interface to occupancy sensing functionality, including configuration and provision of notifications of occupancy status.
