# Nearest Gateway Cluster (0x0024)

## 概述

Nearest Gateway Cluster（Cluster ID `0x0024`）属于 ZCL Rev 7 的 Retail 功能域，章节定位为 14.5。通知或查询最近 gateway 信息以辅助设备通信。

Rev 7 精读范围：`raw/specs/07-5123-07-ZigbeeClusterLibrary_Revision_7-1.pdf p.899-p.903`。本页根据这些页的 cluster identifier、overview、server/client、attributes、commands 和行为说明整理；长表字段仍以来源页为准。
