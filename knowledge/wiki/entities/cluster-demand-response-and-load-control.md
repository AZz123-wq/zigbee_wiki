# Demand Response and Load Control Cluster (0x0701)

## 概述

Demand Response and Load Control Cluster（Cluster ID `0x0701`）属于 ZCL Rev 7 的 Smart Energy 功能域，章节定位为 10.3。发布负荷控制事件，并管理设备对 DRLC 事件的响应。

Rev 7 精读范围：`raw/specs/07-5123-07-ZigbeeClusterLibrary_Revision_7-1.pdf p.613-p.641`。本页根据这些页的 cluster identifier、overview、server/client、attributes、commands 和行为说明整理；长表字段仍以来源页为准。
