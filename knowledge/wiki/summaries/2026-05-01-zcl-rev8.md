## 概述

Zigbee Cluster Library (ZCL) 是 Zigbee 协议栈的应用层核心规范，定义了所有标准 Cluster 的**属性、命令、数据类型和行为**。Rev 8 是当前最新主版本。

## 关键要点

1. ZCL 是 Zigbee 生态中最重要的规范之一，所有设备 Cluster 定义均来自此文档
2. Rev 8 覆盖 15 个功能域；页数基准已修正为 Rev 8 1213 页、Rev 7 929 页，旧 fallback 页数不用于引用
3. 每个 Cluster 定义包括：Cluster ID、Attributes、Commands (Server/Client)、Data Types
4. 与测试规范强关联：每个 Cluster 都有对应的 Test Specification
5. Foundation 章节定义了 ZCL 框架基础（帧格式、寻址、属性报告等）
